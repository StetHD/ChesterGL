Class Detail
chesterGL.PrimitiveBlock(maxPoints, maxLines)
creates a new PrimitiveBlock. All draw functions should be placed in the update method of the primitive block. Every frame, all draw calls are reset. 
var block = new PrimitiveBlock();
block.setUpdate(function () {
	var sz = this.contentSize;
	this.drawPoint(sz.width * 0.5, sz.height * 0.5);
});
Parameters:
{number=} maxPoints
the max number of points, by default it's set to 500
{number=} maxLines
the max number of lines, by default it's set to 500
